# Dataset Information

This project uses the **System Threat Forecaster** dataset from Kaggle.

## Source
https://www.kaggle.com/competitions/System-Threat-Forecaster

## Files Used
- train.csv (not included due to size)
- test.csv (not included)
- sample_submission.csv (not included)

## How to Download
1. Install Kaggle CLI
2. Run:
   kaggle competitions download -c system-threat-forecaster
3. Extract the files into the `data/` directory
